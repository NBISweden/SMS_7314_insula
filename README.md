# Support Request

In brief, the data that we have, derive from the following experimental design:
Brain samples from alcohol dependent patients (N=12 cases) and matched controls (N=12 controls) were first submitted to subcellular fractionations, which yielded three fractions (i.e., synaptosomes, cytosol, insoluble fraction). All three fractions (total N=72 samples) were subjected to TMT-based quantitative proteomics. Data were provided by a commercial company (Creative Proteomics) in .xlsx format. The company also performed and provided results from basic bioinformatic comparisons (inc. differential expression, pathway analysis, etc.), which revealed some statistical significances between cases and controls on the nominal p-level.

We would now like to re-analyze the data and prepare material, e.g., Figures/Tables and Methods, for a publication.

## Requested to do

    - Characterization of subcellular fraction composition
    - Differential abundance (Limma), and pathway/sensitivity analyses
    - Multivariate (plsDA), and pathway analyses
    - Protein redistribution analysis 
    - Cross-fraction integration analyses (DIABLO)
    - Cell-type enrichment and sensitivity analysis
    - Integration with genetic risk loci
    - Proteomic differential alternative splicing analysis
    - Neuronal drug-repurposing analyses


## Data Information

- Type of data: Bulk Proteomics, 3 subcellular compartments
- Code repository: NBISweden/SMS_7314_insula
